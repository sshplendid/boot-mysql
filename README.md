# Spring Boot - RDB

이 소스코드는 스프링 부트에 RDBMS를 연결하는 예제이다.

## 준비사항

### MySQL with Docker

Docker로 MySQL RDBMS 서버를 구동한다.

```bash
cd docker
docker-compose up -d
```
